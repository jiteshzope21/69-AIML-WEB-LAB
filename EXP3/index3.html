HTML Introduction

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser. It is the building block of the web.

A standard HTML document structure looks like this:

<!DOCTYPE html> <html> <body> <h1>Hello World</h1> <p>This is a paragraph.</p> </body> </html>

Key concepts include:

  • Elements: The building blocks (e.g., paragraphs, headings).
  • Tags: The labels used to mark the start and end of an element.
  • Attributes: Provide additional information about elements.
CSS Basics

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in HTML.

CSS handles the look and feel of a webpage. Here is a basic syntax example:

/* Selector { Property: Value; } */ body { background-color: #f0f0f0; font-family: Arial, sans-serif; } h1 { color: blue; text-align: center; }
  • Selectors: Patterns used to select the element(s) you want to style.
  • Properties: Aspects of the element you want to change (e.g., color, width).
JavaScript Overview

JavaScript is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions.

It is the scripting language that enables you to create dynamically updating content, control multimedia, animate images, and much more.

// A simple function function greet(name) { return "Hello, " + name + "!"; } console.log(greet("Developer")); // Output: Hello, Developer!
  • Variables: Containers for storing data values.
  • Functions: Blocks of code designed to perform a particular task.
Flexbox Layout

The Flexible Box Layout Module (Flexbox) makes it easier to design flexible responsive layout structures without using float or positioning.

To start using Flexbox, you need to define a flex container:

.container { display: flex; justify-content: center; /* Aligns horizontally */ align-items: center; /* Aligns vertically */ }
  • Direction: Control if items are in a row or column.
  • Alignment: Easily center items vertically and horizontally.
CSS Grid

CSS Grid Layout is a two-dimensional layout system for the web. It lets you lay out items in rows and columns.

Unlike Flexbox (1-dimensional), Grid handles both dimensions simultaneously:

.grid-container { display: grid; grid-template-columns: auto auto auto; /* 3 Columns */ gap: 10px; /* Space between items */ }
  • Grid Lines: The dividing lines that make up the structure of the grid.
  • Grid Areas: Specific rectangular areas on the grid used to place content.
